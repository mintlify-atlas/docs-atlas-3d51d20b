## Purpose

`CardCrafter` is the primary playlist creation and editing interface. It provides a modal dialog where users can:

- Set playlist title and visual properties
- Add songs via YouTube URLs
- Reorder songs with drag-and-drop
- Import songs from saved slips
- Download playlist as a `.card` file

## Props

<ParamField path="setShowCrafter" type="Function" required>
  Callback to close the crafter modal
</ParamField>

<ParamField path="initVal" type="cardContextType | null" required>

## Key Functions

### YouTube URL Validation

#### `valid(url: string | null): boolean`

Validates YouTube URLs against supported formats:

```typescript
function valid(url: string | null) {
  if (!url || typeof url !== "string") {
    return false;
  }
  
  url = url.trim();
  
  const patterns = [
    /youtube\.com\/watch\?v=[a-zA-Z0-9_-]{11}/,
    /youtu\.be\/[a-zA-Z0-9_-]{11}/,
    /youtube\.com\/embed\/[a-zA-Z0-9_-]{11}/,
    /m\.youtube\.com\/watch\?v=[a-zA-Z0-9_-]{11}/,
    /music\.youtube\.com\/watch\?v=[a-zA-Z0-9_-]{11}/,
  ];
  
  return patterns.some((pattern) => pattern.test(url));
}
```

#### `getID(url: string | null): string | null`

Extracts 11-character YouTube video ID from various URL formats:

```typescript
const getID = (url: string | null) => {
  if (typeof url !== "string") {
    return null;
  }
  url = url.trim();
  
  // Match standard watch URL
  let match = url.match(/[?&]v=([a-zA-Z0-9_-]{11})/);
  if (match) {
    return match[1];
  }
  
  // Match short URL
  match = url.match(/youtu\.be\/([a-zA-Z0-9_-]{11})/);
  if (match) {
    return match[1];
  }
  
  // Match embed URL
  match = url.match(/youtube\.com\/embed\/([a-zA-Z0-9_-]{11})/);
  if (match) {
    return match[1];
  }
  
  return null;
};
```

## Drag and Drop Implementation

CardCrafter uses `@dnd-kit` for song reordering:

```tsx
import { closestCorners, DndContext, type DragEndEvent } from "@dnd-kit/core";
import {
  SortableContext,
  useSortable,
  verticalListSortingStrategy,
} from "@dnd-kit/sortable";

const handleDragEnd = (event: DragEndEvent) => {
  const { active, over } = event;
  if (!over || active.id === over.id) return;
  
  setCrafterContext((ctx) => {
    const oldIndex = ctx.inputCard.songs.findIndex((s) => s.id === active.id);
    const newIndex = ctx.inputCard.songs.findIndex((s) => s.id === over.id);
    const newSongs = [...ctx.inputCard.songs];
    newSongs.splice(oldIndex, 1);
    newSongs.splice(newIndex, 0, ctx.inputCard.songs[oldIndex]);
    return {
      ...ctx,
      inputCard: { ...ctx.inputCard, songs: newSongs },
    };
  });
};
```

## Song Management

### Adding Songs

Users can add songs by:

1. **Manual Entry**: Paste YouTube URL and enter title
2. **Auto-Fetch**: Click the fetch button to retrieve title from YouTube
3. **From Slips**: Import from saved playlists via `AddSlips` component

```tsx
<button
  onClick={() => {
    if (!valid(crafterContext.link) || !crafterContext.title) {
      window.alert("Invalid Link / No Title");
      return;
    }
    
    // Check if song already exists
    const existingSongIndex = crafterContext.inputCard.songs.findIndex(
      (song) => song.id === getID(crafterContext.link)
    );
    
    if (existingSongIndex !== -1) {
      // Update existing song
      const updatedSongs = [...crafterContext.inputCard.songs];
      updatedSongs[existingSongIndex] = {
        title: crafterContext.title,
        id: getID(crafterContext.link) || "",
      };
      setCrafterContext({
        ...crafterContext,
        inputCard: { ...crafterContext.inputCard, songs: updatedSongs },
      });
    } else {
      // Add new song
      setCrafterContext({
        ...crafterContext,
        inputCard: {
          ...crafterContext.inputCard,
          songs: [...crafterContext.inputCard.songs, {
            title: crafterContext.title,
            id: getID(crafterContext.link) || "",
          }],
        },
      });
    }
  }}
>
  <Plus size={35} />
</button>
```

## Duplicate Prevention

CardCrafter automatically removes duplicate songs on mount and update:

```tsx
useEffect(() => {
  setCrafterContext((prev) => {
    const uniqueSongsMap = new Map();
    const filteredSongs = prev.inputCard.songs.filter((song) => {
      if (uniqueSongsMap.has(song.id)) return false;
      uniqueSongsMap.set(song.id, true);
      return true;
    });
    
    if (filteredSongs.length !== prev.inputCard.songs.length) {
      return {
        ...prev,
        inputCard: {
          ...prev.inputCard,
          songs: filteredSongs,
        },
      };
    }
    
    return prev;
  });
}, [crafterContext.inputCard.songs]);
```
