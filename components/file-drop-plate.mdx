## Purpose

`FileDropPlate` provides a drag-and-drop interface for importing `.card` files:

- Accept file drops
- Browse for files via click
- Parse JSON card data
- Load card into application context
- Visual feedback during drag

### onDrop Handler

Processes dropped files:

```tsx
const onDrop = useCallback((receivedCard: File[]) => {
  if (receivedCard.length > 1) return;  // Only accept single files
  
  receivedCard.forEach((file) => {
    const reader = new FileReader();
    
    reader.onabort = () => console.log("file reading was aborted");
    reader.onerror = () => console.log("file reading has failed");
    
    reader.onload = () => {
      const text: string = String(reader.result);
      const parsedText: typeof appContext.loadedCard = JSON.parse(text);
      setAppContext((prev: typeof appContext) => ({
        ...prev,
        currentTrack: 0,
        ...parsedText,
      }));
    };
    
    reader.readAsText(file);
  });
}, []);
```

## File Validation

Only single files are accepted:

```tsx
if (receivedCard.length > 1) return;
```

This prevents issues with multiple simultaneous imports.

## JSON Parsing

The component expects `.card` files with this structure:

```json
{
  "loadedCard": {
    "properties": {
      "title": "Playlist Name",
      "color": "#3E3F76",
      "blur": "1",
      "bgType": "color",
      "cdHero": "char",
      "char": "🎵"
    },
    "songs": [
      {
        "title": "Song Title",
        "id": "YouTubeVideoID"
      }
    ]
  }
}
```

The parsed data is spread into the app context:

```tsx
setAppContext((prev: typeof appContext) => ({
  ...prev,
  currentTrack: 0,  // Reset to first track
  ...parsedText,    // Spread parsed card data
}));
```

## Context Access

The component accesses the `LoadedCard` context:

```tsx
import { LoadedCard } from "@/services/ContextService";

const { appContext, setAppContext } = useContext(LoadedCard);
```

This allows it to update the application state with loaded card data.

<Info>
  FileDropPlate must be wrapped in a `ContextProvider` to function properly.
</Info>

## Integration Example

```tsx
import { FileDropPlate } from '@/components/ui/FileDropPlate/FileDropPlate';
import ContextProvider from '@/services/ContextService';
import { useState } from 'react';

function App() {
  const [pulled, setPulled] = useState(false);
  
  return (
    <ContextProvider>
      <FileDropPlate
        pulled={pulled}
        setPulled={setPulled}
      />
    </ContextProvider>
  );
}
```

## Best Practices

1. **Validation**: Add JSON schema validation for imported cards
2. **Error Messages**: Show user-friendly errors for invalid files
3. **Loading State**: Display loading indicator during file parsing
4. **File Type**: Restrict to `.card` or `.json` extensions
5. **Large Files**: Add file size limits

```tsx
const { getRootProps, getInputProps, isDragActive, open } = useDropzone({
  onDrop,
  noClick: true,
  accept: {
    'application/json': ['.card', '.json']
  },
  maxSize: 1048576  // 1MB
});
```
