Welcome to Ode! This guide will walk you through creating your first music card, adding songs, and customizing your card's appearance.

<Steps>
  <Step title="Access the application">
    Open Ode in your desktop browser. The application requires a desktop environment and is not available on mobile devices.
    
    When you first launch Ode, you'll see the main interface with two buttons at the bottom left:
    - **Create**: Start a new card from scratch
    - **Edit**: Load and modify an existing card file
    
    <Tip>
      Press `c` on your keyboard to quickly open the card creator at any time.
    </Tip>
  </Step>

  <Step title="Create your first card">
    Click the **Create** button to open the Card Crafter interface. You'll see three main sections:
    
    1. **Title section** (top left): Enter a name for your card
    2. **Card preview** (bottom left): See your card design in real-time
    3. **Song management** (right side): Add and organize your songs
    
    Start by entering a title for your card in the top input field. This will be the name of your music collection.
    
    <Tip>
      The card preview shows how many songs you've added and updates in real-time as you make changes.
    </Tip>
  </Step>

  <Step title="Add songs using YouTube links">
    Ode uses YouTube as its music source. Here's how to add songs:
    
    1. **Copy a YouTube URL** from any video
    2. **Paste it into the link field** in the Card Crafter
    3. **Click the download arrow button** to automatically fetch the video title, or enter a custom title manually
    4. **Click the + button** to add the song to your card
    
    ### Supported YouTube URL formats
    
    Ode parses and extracts video IDs from these URL patterns:
    
    ```typescript
    // Standard YouTube watch URLs
    youtube.com/watch?v=VIDEO_ID
    
    // Short URLs
    youtu.be/VIDEO_ID
    
    // Embed URLs
    youtube.com/embed/VIDEO_ID
    
    // Mobile URLs
    m.youtube.com/watch?v=VIDEO_ID
    
    // YouTube Music URLs
    music.youtube.com/watch?v=VIDEO_ID
    ```
    
    The URL parsing logic extracts the 11-character video ID from any of these formats:
    
    ```typescript
    // Example: getID function (from CardCrafter.tsx:78-104)
    const getID = (url: string | null) => {
      url = url.trim();
      
      // Matches ?v=VIDEO_ID pattern
      let match = url.match(/[?&]v=([a-zA-Z0-9_-]{11})/);
      if (match) return match[1];
      
      // Matches youtu.be/VIDEO_ID pattern
      match = url.match(/youtu\.be\/([a-zA-Z0-9_-]{11})/);
      if (match) return match[1];
      
      // Additional patterns for embed, mobile, and music URLs...
      return null;
    };
    ```
    
    <Tip>
      You can reorder songs by dragging and dropping them in the list. Hover over a song title to see the full name if it's truncated.
    </Tip>
  </Step>

  <Step title="Customize your card appearance">
    Use the **Color Change Option** controls to personalize your card:
    
    - **Background Type**: Choose between a solid color or a blurred thumbnail from one of your songs
    - **Background Color**: Select any color for your card background
    - **Hero Display**: Choose between showing an emoji character or your card title
    - **Character/Emoji**: Select an emoji to represent your card
    
    Click the card preview to flip it and see the song list on the back side.
    
    <Tip>
      The card preview shows both the front and back of your card. Click the preview to rotate it 180 degrees and inspect the tracklist.
    </Tip>
  </Step>

  <Step title="Download and share your card">
    Once you're satisfied with your card:
    
    1. **Ensure you have a title** and at least one song added
    2. **Click the Download button** to save your card
    3. Your card will be saved as a `.card` file with your chosen title
    
    ### Card file format
    
    Ode cards are saved as JSON files with the following structure (from CardCrafter.tsx:205-219):
    
    ```json
    {
      "loadedCard": {
        "properties": {
          "title": "My Awesome Playlist",
          "color": "#11262A",
          "blur": "1",
          "bgType": "color",
          "cdHero": "char",
          "char": "📻"
        },
        "title": "My Awesome Playlist",
        "songs": [
          {
            "title": "Song Title",
            "id": "VIDEO_ID"
          }
        ]
      }
    }
    ```
    
    You can share this `.card` file with others, and they can load it by clicking the **Edit** button on the main interface.
    
    <Tip>
      Cards must have at least one song and a title to be downloaded. The application will alert you if either is missing.
    </Tip>
  </Step>
</Steps>
